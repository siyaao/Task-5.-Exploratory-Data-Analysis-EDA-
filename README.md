Titanic Dataset - Exploratory Data Analysis (EDA)

- Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python. The objective is to understand the dataset, identify patterns, analyze relationships between variables, and extract meaningful insights regarding passenger survival.

- Tools and Libraries

 Python
 Pandas
 Matplotlib
 Seaborn
 Jupyter Notebook

- Dataset

The Titanic dataset contains passenger information such as:
 Passenger Class (Pclass)
 Gender (Sex)
 Age
 Fare
 Embarked Port
 Survival Status

- Analysis Performed

 Dataset exploration using `head()`, `info()`, and `describe()`
 Missing value analysis
 Survival distribution analysis
 Gender-wise survival analysis
 Passenger class analysis
 Age and fare distribution analysis
 Correlation analysis using heatmaps
 Relationship analysis using pairplots and scatterplots

- Key Findings

 Female passengers had higher survival rates than male passengers.
 First-class passengers were more likely to survive.
 Most passengers were between 20 and 40 years old.
 Higher ticket fares were associated with better survival chances.
 Gender and passenger class were important factors influencing survival.

- Project Files

  Titanic_EDA.ipynb – Jupyter Notebook containing EDA code and visualizations
  titanic.csv – Dataset used for analysis
  EDA_Report.pdf – Detailed report of findings
  README.md – Project documentation

- Outcome

This project demonstrates the use of Exploratory Data Analysis techniques to understand data, discover trends, and derive meaningful insights from real-world datasets.
